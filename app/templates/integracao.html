Sua URL de ingestão

Cole esta URL no app Health Auto Export (iPhone), numa automação de REST API exportando Workouts em JSON. Os treinos passam a chegar sozinhos.

O token identifica a sua conta. Trate como senha — quem tiver essa URL consegue enviar treinos para você.

Como configurar (resumo)

  1. No app Zepp: Profile → Add accounts → Apple Health (sincroniza os treinos do relógio para o Apple Health).
  2. No Health Auto Export (licença Basic): crie uma Automation → tipo REST API.
  3. Método POST, formato JSON, dados Workouts (com Route + Heart Rate), e cole a URL acima.
  4. Opcional: em vez do ?token= na URL, use o endpoint {{ ingest_endpoint }} com o header Authorization: Bearer SEU_TOKEN.
